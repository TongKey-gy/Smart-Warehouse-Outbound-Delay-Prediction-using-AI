# Smart Warehouse Outbound Delay Prediction using AI

이 프로젝트는 데이콘 `스마트 창고 출고 지연 예측 AI 경진대회` 참가를 위해 구축한 머신러닝 프로젝트입니다. 스마트 물류 운영 데이터에 기반해 향후 출고 지연을 예측하고, 창고 운영 효율화에 활용할 수 있는 예측 모델을 개발하는 것을 목표로 합니다.

## Competition Overview

본 프로젝트는 **스마트 창고 출고 지연 예측 AI 경진대회**를 기반으로 진행되었습니다.  
해당 대회는 스마트 창고 운영 데이터를 활용하여 **향후 30분 동안 발생할 평균 출고 지연 시간**을 예측하는 **회귀(Regression) 문제**입니다.

### 📦 대회 배경

데이터는 **AMR(Autonomous Mobile Robot)** 기반 스마트 물류창고 시뮬레이션 환경에서 생성되었으며, 실제 물류 창고에서 발생할 수 있는 다양한 병목 상황을 반영하도록 설계되었습니다.

### 🚀 프로젝트 목표

본 프로젝트는 시계열 기반 창고 운영 데이터를 활용하여  
**물류 병목을 사전에 감지하고 출고 지연을 정량적으로 예측하는 모델**을 개발하는 것을 목표로 합니다.

### 📊 데이터 구성

- 총 **12,000개**의 독립적인 창고 운영 시나리오
- 각 시나리오는 약 **6시간** 운영 데이터 포함
- **15분 간격**으로 구성된 **25개 타임슬롯**
- 각 시점마다 창고 상태를 **스냅샷 형태**로 제공

### 🔍 주요 입력 변수

다음과 같은 다양한 운영 지표가 포함됩니다:

- 로봇 상태 (AMR 운영 정보)
- 주문 유입량
- 배터리 및 충전 상태
- 통로 혼잡도
- 기타 물류 운영 관련 변수

이러한 변수들의 **복합적인 상호작용**을 통해 출고 지연 패턴을 모델링해야 합니다.

### 🎯 예측 목표

- **Target 변수:** `avg_delay_minutes_next_30m`
- 향후 **30분 평균 출고 지연 시간** 예측
