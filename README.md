# Smart Warehouse Outbound Delay Prediction using AI

이 프로젝트는 데이콘 `스마트 창고 출고 지연 예측 AI 경진대회` 참가를 위해 구축한 머신러닝 프로젝트입니다. 스마트 물류 운영 데이터에 기반해 향후 출고 지연을 예측하고, 창고 운영 효율화에 활용할 수 있는 예측 모델을 개발하는 것을 목표로 합니다.

## Competition Overview

이 대회는 스마트 창고 운영 시점의 데이터를 바탕으로 **향후 30분 동안 발생할 평균 출고 지연 시간**을 예측하는 회귀 문제입니다. 데이터는 **AMR(Autonomous Mobile Robot) 기반 스마트 물류창고 시뮬레이션 환경**에서 생성되었으며, 실제 운영에서 발생할 수 있는 복합적인 병목 상황을 반영하도록 설계되어 있습니다.

학습 데이터는 **12,000개의 독립적인 창고 운영 시나리오**로 구성되어 있고, 각 시나리오는 **약 6시간 동안의 운영 흐름**을 담고 있습니다. 시간 축은 **15분 간격의 25개 타임슬롯**으로 구성되며, 각 시점마다 창고 상태와 운영 지표가 스냅샷 형태로 제공됩니다.

주요 입력 변수에는 **로봇 상태**, **주문량**, **배터리 및 충전 상태**, **통로 혼잡도**를 포함한 다양한 운영 정보가 포함되어 있으며, 이들 변수의 상호작용을 통해 출고 지연 패턴을 모델링해야 합니다. 예측 대상 타깃 변수는 `avg_delay_minutes_next_30m`이며, 메인 데이터 외에도 창고 구조 정보를 담은 **추가 테이블 `layout_info.csv`**가 함께 제공됩니다.

본 프로젝트는 이러한 시계열·운영 데이터를 기반으로 물류 병목을 사전에 감지하고, 출고 지연을 정량적으로 예측할 수 있는 모델링 전략을 연구하는 데 초점을 둡니다.
